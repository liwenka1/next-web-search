# 一个简单好用的起始页

## 前言

经常使用搜索引擎的时候会感到非常华丽，但实际我并用不到那么多功能，于是开发了这个非常简洁的起始页

### 技术选型

- next.js + tailwindcss + shadcn/ui + magic-ui + aceternity-ui + motion

## 功能介绍

- 现代化UI设计和动画
- 移动端适配
- 支持日夜颜色模式切换
- 支持跳转翻译页面
- 搜索框联想搜索
- 支持搜索引擎切换
- 支持 Vercel 直接部署上线
